The Playfair cipher is a classical symmetric encryption technique that encrypts pairs of letters (digraphs) instead of single letters. It uses a 5√ó5 matrix constructed from a keyword (with I/J usually combined)

#### <ins>Example:</ins>
Plaintext: `balloon` <br>
Key: `monarchy` <br>
Matrix Size: `5 √ó 5`

<br>
